# **App Name**: Gotcha 2.0

## Core Features:

- Vybz Feed: A home feed displaying short videos in a fullscreen vertical scroll.
- Profile Page: A personalized space for users to showcase their identity and content.
- Chat Page: Real-time messaging with message bubbles, input bar, and typing indicators.
- AI Chat Assistant: Optional AI assistant that users can tool to answer questions.
- Explore Page: A discovery engine to find creators, videos, trends, and topics via search and category chips.
- Create Page: A space for users to record or upload short videos with filters and captioning.
- Gamified Reward System: A gamified system that reward interactions.

## Style Guidelines:

- Primary color: Lavender (#C9A7EB) for a calming and elegant feel.
- Secondary color: Sky Blue (#A1C4FD) complements the lavender for a gradient effect.
- Accent color: Deep Purple (#6A5ACD) to highlight interactive elements.
- Font pairing: 'Poppins' (sans-serif) for headers and 'Inter' (sans-serif) for body text, using Regular and Medium weights. Note: currently only Google Fonts are supported.
- Use Lucide Icons: minimal, line-based icons throughout the UI.
- Implement rounded corners, soft shadows, and glassmorphism blur effects for UI elements.
- Use a floating bottom navigation bar for easy access to main sections.
- Apply smooth transitions and subtle micro-animations for enhanced user experience.
